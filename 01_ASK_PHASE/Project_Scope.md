# Project Scope
## Cyclistic Bike-Share Case Study - Ask Phase

---

## Project Definition

**Project Name:**  
Cyclistic Member Conversion Analysis – Understanding User Behavioral Patterns

**Project Duration:**  
One week (ASK phase followed by Prepare → Process → Analyze → Share → Act phases)

**Project Type:**  
Marketing Analytics / Customer Segmentation Analysis

---

## Primary Objective

Analyze 12 months of historical Cyclistic bike-share trip data to identify key behavioral differences between annual members and casual riders, providing data-driven insights to support the design of marketing strategies aimed at converting casual riders into annual members.

---

## Scope Boundaries

### IN SCOPE - What We're Analyzing:

**Data to Include:**
- 12 months of historical Cyclistic trip data
- Ride duration and frequency patterns
- Temporal patterns (day of week, seasonal variations)
- Rider type classification (member vs. casual rider)
- Trip characteristics (start/end locations, bike type)

**Analysis Focus:**
- Behavioral differences between customer segments
- Usage frequency and duration metrics
- Temporal patterns and preferences
- Seasonal trends and variations
- Statistical comparisons between groups

**Deliverables Included:**
1. Clear business task statement
2. Comprehensive data source documentation
3. Detailed data cleaning and manipulation documentation
4. Analysis summary with key metrics and findings
5. Professional data visualizations (4-6 charts minimum)
6. Top 3 actionable recommendations for marketing strategy
7. Complete portfolio-ready case study documentation

### OUT OF SCOPE - What We're NOT Analyzing:

**Explicitly Excluded:**
- Customer personal identifiable information (PII) – prohibited by data privacy policy
- Credit card or payment information
- Customer residential location data
- Individual customer tracking or profiling
- Predictive modeling or forecasting
- Customer satisfaction surveys or qualitative feedback
- Marketing channel effectiveness (covered by secondary questions)
- Digital media ROI analysis (separate future project)
- Customer demographics beyond member/casual classification

**Note:** These out-of-scope items may be addressed by follow-up analyses after this primary research is completed.

## Project Phases & Timeline

### Phase 1: ASK
- Define business task and key questions
- Conduct stakeholder analysis
- Establish project scope and success criteria
- **Deliverable:** Business Task Statement, Key Questions, Stakeholder Analysis, Project Scope

### Phase 2: PREPARE
- Download and organize 12 months of trip data
- Assess data quality and credibility (ROCCC framework)
- Document data sources and structure
- Address data privacy and security protocols
- **Deliverable:** Data source documentation

### Phase 3: PROCESS
- Check data for errors and inconsistencies
- Clean and transform data for analysis readiness
- Create calculated fields (ride_length, day_of_week)
- Combine multiple data files into unified dataset
- **Deliverable:** Detailed cleaning documentation

### Phase 4: ANALYZE
- Conduct descriptive statistical analysis
- Create pivot tables for comparison analysis
- Calculate key metrics (mean, max, mode, averages by segment)
- Identify trends and relationships
- **Deliverable:** Analysis summary with findings

### Phase 5: SHARE
- Design and create professional visualizations
- Develop compelling visual narratives
- Create presentation-ready charts and graphs
- Prepare executive summary
- **Deliverable:** Supporting visualizations and key findings

### Phase 6: ACT
- Develop top 3 recommendations based on analysis
- Create portfolio presentation
- Prepare case study narrative
- Complete project documentation
- **Deliverable:** Portfolio case study with recommendations
